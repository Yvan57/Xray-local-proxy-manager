# Xray-local-proxy-manager
⚡ Xray Local Proxy Manager - удобный менеджер для управления несколькими локальными SOCKS5 прокси одновременно. Поддерживает VLESS, VMess, Trojan и Shadowsocks. Включает: проверку IP через прокси, реальный мониторинг активности с логом соединений, автосохранение конфигурации, диагностику проблем.
